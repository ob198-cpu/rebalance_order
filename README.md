# rebalance_order

NISA Alpha 運用HTMLレポートの公開用リポジトリです。

## HTML

- `reports/rebalance_order.html`  
  四半期・年次の発注内容。これだけで発注内容を確認できます。
- `reports/reload_monitor.html`  
  日次のリロード枠トリガー判定。
- `reports/monthly_report.html`  
  月次の3指数比較・チェックリスト。

`index.html` から3本へリンクしています。GitHub Pagesを有効化する場合は、`main` ブランチの root を公開元にしてください。

## 注意

このHTMLは運用補助資料であり、投資助言や自動売買システムではありません。発注前に価格、株数、口座区分、NISA残枠を必ず確認してください。
